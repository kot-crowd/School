# Программа для проверки оценок

## Описание проекта

Эта программа позволяет пользователю вводить баллы студента по трем предметам: математике, физике и химии. На основе введенных данных программа определяет оценку по каждому предмету и вычисляет общий и средний баллы. Если средний балл превышает 80, студент переводится на следующий курс, в противном случае ему требуются дополнительные занятия.

1. Убедитесь, что Python установлен на вашем компьютере. Если нет, скачайте и установите его с [официального сайта Python](https://www.python.org/).
2. Скачайте или клонируйте репозиторий с программой.
3. Откройте терминал или командную строку.
4. Перейдите в директорию, где находится файл `test.py`.
5. Запустите программу командой:
